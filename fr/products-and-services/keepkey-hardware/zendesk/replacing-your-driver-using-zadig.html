Remplacer votre pilote avec Zadig

Votre ordinateur Windows ne détecte pas votre KeepKey, Trezor ou Ledger ?

Nous avons constaté que certaines machines n'ont pas les paramètres de pilote par défaut appropriés pour reconnaître avec succès un dispositif matériel de porte-monnaie sur beta.shapeshift.com. Ne vous inquiétez pas. Nous avons une solution directement issue de Windows pour vous aider.  Ce qui suit explique comment utiliser un installateur de pilote USB appelé Zadig.
 
Installation de Zadig

1. Pour commencer, allez sur Zadig. Cliquez sur "Zadig 2.4" et commencez l'installation.

2. Une fois installé, vous devriez voir quelque chose comme ceci :

Remplacer votre pilote

3. Cliquez sur "Options" et sélectionnez "List All Devices" (Lister tous les périphériques).

4. Vous devriez maintenant voir que les périphériques ont été détectés. Cliquez sur la liste déroulant et sélectionnez "KeepKey Interface (Interface 0)". Pour cette procédure, il est important de noter que nous n'allons PAS sélectionner l'interface KeepKey U2F (interface 1).

5. À droite de la flèche verte, utilisez la ou les flèches vers le haut ou vers le bas pour naviguer vers “libusbK (v3.0.7.0)”.

 

Remarque: Si votre pilote actuel est déjà “libusbK (v3.0.7.0)”, utilisez la ou les flèches vers le haut ou vers le bas pour sélectionner “WinUSB (v6.1.7600.16385)”.

Zadig devrait maintenant ressembler à quelque chose comme ceci :

6. Cliquez sur "Replace Driver" (Remplacer le pilote).

7. Le pilote va commencer à s'installer.

8. Vous devriez maintenant voir que votre pilote actuel a changé pour “libusbK” (ou “WinUSB” selon celui que vous avez remplacé).

Si vous aviez beta.shapeshift.com ouvert, allez-y et fermez-le. Redémarrez l'ordinateur auquel vous avez essayé de connecter votre appareil et essayez d'appairer votre KeepKey. Si le pilote était effectivement la cause de la non-détection de votre appareil, vous devriez maintenant voir que vous pouvez connecter votre KeepKey.

Si vous avez des difficultés à remplacer votre pilote ou si vous souhaitez obtenir de l'aide pour inverser ce processus, veuillez contacter notre équipe d'assistance.
