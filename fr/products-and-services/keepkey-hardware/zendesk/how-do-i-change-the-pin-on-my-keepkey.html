Comment puis-je changer le PIN/NIP de ma clé KeepKey ?

Vous devrez peut-être modifier votre code NIP de temps en temps pour rester en sécurité ! N'oubliez pas que si vous oubliez votre code NIP, vous devrez effacer complètement l'appareil pour le réinitialiser. Pour suivre les instructions ci-dessous, vous devez connaître votre code PIN actuel avant de le modifier. Si vous avez oublié votre code NIP et que vous ne pouvez pas accéder à votre KeepKey, suivez les étapes décrites dans cet article. 

1. Assurez-vous que votre KeepKey est branché et connecté à beta.shapeshift.com.

2. Accédez à la page des paramètres en cliquant sur le menu déroulant en haut à droite de la plate-forme et en cliquant sur “Settings” (Paramètres). Ou vous pouvez cliquer sur ce lien.

3. Sous “Linked Wallet” (Porte-monnaie Lié), cliquez sur "Change PIN" (Modifier le code NIP).

4. Confirmez que vous souhaitez modifier votre code NIP en maintenant le bouton enfoncé sur votre KeepKey.

5. Entrez votre NIP actuel.

5a. Veillez à utiliser correctement le cryptogramme lorsque vous saisissez votre code NIP.

Exemple: Si vous voulez choisir le nombre "6", vous devez cliquer sur la case en haut à gauche.

6. Entrez  votre nouveau NIP.

7. Entrez votre nouveau NIP à nouveau (n'oubliez pas que les chiffres sont maintenant brouillés et qu'ils se trouvent à des endroits différents de ceux où vous avez entré votre nouveau code NIP la première fois).

 8. Débranchez la clé KeepKey et rebranchez-la. Le tour est joué ! Vous avez réussi à changer votre code PIN !
