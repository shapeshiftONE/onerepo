Cómo utilizar el KeepKey Bridge

¿Qué es KeepKey Bridge?

KeepKey Bridge es una aplicación de Microsoft Windows que está diseñada para ayudar con los problemas de conectividad webUSB entre KeepKey y una máquina con Windows.

Bridge se ejecuta como una aplicación que abre una conexión USB local a KeepKey. Bridge expone un puerto de host local que la aplicación web ShapeShift puede usar para comunicarse con KeepKey. KeepKey en sí no es consciente del Bridge, pero la aplicación web ShapeShift debe estar dirigida para usar Bridge.

¿Cuándo debería intentarlo?

Si está utilizando una máquina con Windows y tiene problemas para conectar su KeepKey a beta.shapeshift.com, intente ejecutar Bridge siguiendo las instrucciones a continuación.

Instalar The Bridge

1. Antes de usar The Bridge, asegúrese de que su KeepKey esté actualizado al cargador de arranque / firmware más reciente. Se puede encontrar la aplicación de actualización here. Descargue la última versión de KeepKey bridge here.

Nota: Es probable que su computadora le dé algunas advertencias al descargar / ejecutar esta aplicación. Puede hacer clic en "Ejecutar de todos modos" para continuar con la instalación.

2. Una vez que se descarga el archivo kkbsetup.exe, puede verificar que el archivo es lo que afirmamos comparando el SHA256 del ejecutable con el resumen publicado en las notas de la versión. Esto se hace abriendo una ventana de cmd (símbolo del sistema), navegando a la carpeta Descargas y escribiendo: cetuil -hashfile kkbsetup.exe SHA256. La información resultante que se muestre debe coincidir con la publicada en el python-keepkey releases notas. Si todo coincide, haga doble clic en el archivo para instalar (se recomienda crear un acceso directo en el escritorio para su comodidad).

Usar The Bridge

3. Al iniciar la aplicación KeepKey Bridge, aparecerá una ventana en blanco. Aunque está en blanco, el puente se está ejecutando (sabemos que no es una interfaz de usuario ideal, ¡pero estamos trabajando en una mejora pronto!). Cuando la ventana está cerrada, The Bridge dejará de funcionar.

4. Con The Bridge en funcionamiento, dirígete a beta.shapeshift.com. Seleccione KeepKey como la billetera que le gustaría emparejar.

5. Haga clic en la casilla de verificación y luego seleccione "PROBAR KEEPKEY BRIDGE" (TRY KEEPKEY BRIDGE).

Si tiene éxito, su billetera comenzará a sincronizarse y eventualmente se conectará a la plataforma.

Solución de problemas

Si se encuentra con algún mensaje de error (como el que se muestra arriba) al intentar emparejar su KeepKey a través del puente o simplemente no se conecta, intente lo siguiente:

1. Vaya a settings page y "Olvide" (forget) su KeepKey.

2. Desconecte su KeepKey.

3. Actualice beta.shapeshift.com.

4. Cierren la aplicacion The bridge.

5. Repita los pasos de la sección "Uso del puente" (Using The Bridge).

 

Si tiene alguna pregunta, comuníquese con el servicio de atención al cliente here.
